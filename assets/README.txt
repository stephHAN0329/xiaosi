# Game Resource Files

To make the game run properly, you need to add the following resource files to this directory:

## Image Files

1. `pixel-bg.png` - Pixel art background image
2. `character.png` - Game character image
3. `character-sad.png` - Character image for game over screen

## Audio Files

1. `background-music.mp3` - Background music
2. `correct.mp3` - Sound effect for correct answers
3. `wrong.mp3` - Sound effect for wrong answers
4. `game-over.mp3` - Sound effect for game over
5. `button.mp3` - Sound effect for button clicks

You can get free pixel art resources from the following websites:

- Image resources:
  - https://itch.io/game-assets/free/tag-pixel-art
  - https://opengameart.org/art-search-advanced?keys=pixel
  - https://www.kenney.nl/assets

- Sound effect resources:
  - https://freesound.org/
  - https://opengameart.org/art-search-advanced?keys=sound
  - https://www.zapsplat.com/

Please make sure the resources you use are free or you have the rights to use them.
